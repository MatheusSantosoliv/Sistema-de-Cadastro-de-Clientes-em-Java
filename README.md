# Sistema-de-Cadastro-de-Clientes-em-Java
Sistema de Cadastro de Clientes em Java, desenvolvido para praticar lógica de programação, orientação a objetos e operações CRUD. A aplicação permite cadastrar, listar, buscar, atualizar e remover clientes por meio de um menu interativo no terminal, simulando uma rotina comum de sistemas administrativos e comerciais.
