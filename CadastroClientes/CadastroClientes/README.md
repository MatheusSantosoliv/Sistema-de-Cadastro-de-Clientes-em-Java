# Sistema de Cadastro de Clientes em Java

## Descrição
Projeto desenvolvido em Java com o objetivo de simular um sistema de cadastro de clientes. A aplicação permite cadastrar, listar, buscar, atualizar e remover clientes por meio de um menu interativo no terminal.

## Funcionalidades
- Cadastrar clientes
- Listar todos os clientes
- Buscar cliente por nome
- Atualizar dados de um cliente
- Remover cliente pelo ID

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- Scanner para entrada de dados

## Estrutura do projeto
- `Cliente.java` → classe que representa os dados do cliente
- `ClienteService.java` → classe responsável pelas operações do sistema
- `Main.java` → classe principal com menu interativo

## Objetivo do projeto
Praticar conceitos de lógica de programação, orientação a objetos, organização de código e operações CRUD, aplicando esses conhecimentos em um sistema simples com contexto administrativo.
